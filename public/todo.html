Todo List App

Make a list of things

Make a todo list and check off the items once they're completed. If you've used a todo list or made a grocery list, you know how this works! You can even delete the list once you're done with it.

This was one of the first Ember apps I built. I built a JSON API to support it using CandyCane.js. The list titles have hover effects so the user can tell which list is active. The list titles are scrollable and disappear behind the "Add List" button, so that the button is never overed up.
