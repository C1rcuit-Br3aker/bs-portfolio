Chatty

A 90's style chatroom

This is an old school style chatroom. The room can only be accessed by having a login. Trying to access the room without logging in sends you to the login screen. Being fan of the Nashville Predators and hockey in general, I felt my chatroom should refelct that. If you don't want to create your own login, use email@email.com, password is "password". Feel free to type whatever you want!

It is built with JavaScript, Ember, HTML, and CSS. I had to set up a simple back end models using SLC Loopback. Messages are assigned a timestamp from the server. The chat window refeshes and auto scrolls to the bottom as a way of ensuring the latest messages are shown. Only 3 of my other classmates completed this assignment and it was fairly popular amongst us for a few days. Building a chatroom for the first time was an enjoyable challenge.
